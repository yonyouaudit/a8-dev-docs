# 欢迎使用 A8 数智化审计平台 开发者文档
 
## 1. 平台介绍

**审友A8数智化审计平台** (以下简称 *平台*)是 **审友数治科技** 开发的 `审计行业` 一体化平台，基于该平台，可以通过 `搭建模式` 构建各种审计类业务系统，也可以通过 `连接模式` 与其他系统进行集成，实现审计业务的数字化、智能化、一体化。

## 2. 平台特点

- **低代码**：平台提供 `低代码` 的开发模式，通过拖拽组件、配置属性、编写脚本，即可完成业务系统的开发。
- **一体化**：平台提供 `一体化` 的开发模式，通过配置数据源、配置数据模型、配置业务模型，即可完成业务系统的开发。
- **开放性**：平台提供完善的 `二次开发` 支持，通过系统提供的前后端 `API` 接口及完善的文档可以快速进行系统扩展类开发。

## 3. 平台架构

![产品架构](./attachs/arch.png)

平台属于典型的 `微服务` 架构，`前后端分离` 模式，支持 `容器化` 部署。

- 表单引擎
- 流程引擎
- 查询分析引擎
- 数据集成引擎
- 文档引擎
- 任务引擎
- 检索引擎
