# aip-modelview

AIP-Framework ModelVeiw 组件，提供两个主要组件

## ModelView

提供模型列表数据展现，图形展现功能。

## QueryModel

提供完整模型展现：包含自定义菜单、参照搜索、模型列表数据展现 (该组件也是默认导出组件)。

## 构建组件

```bash
export NODE_OPTIONS=--openssl-legacy-provider
npm install --legacy-peer-deps
npm run build
```

## 更新 `2022-11-30`

1. 增加模型穿透配置支持静态参数替换（仅支持URL穿透，参数名称需要保持一致，#也需要存在#）

    1. 在URL中，配置#xxx#（可多次重复使用，参数名称需不同）内容
    2. 在目标参数中左侧添加上方参数名称，右侧选择字段内容
    3. 穿透时会将对应的实际值对参数名称进行替换

示例 

 如配置穿透路由为`http://www.xxxx.com/#test#`
 ,目标参数为测试（实际值为yonyou），则会将测试对应的实际值与`#test#`
 进行替换,则最终的URL为：`http://www.xxxx.com/yonyou`
