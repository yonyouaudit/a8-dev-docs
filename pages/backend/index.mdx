# A8 产品技术栈说明

A8 产品是全新开发的产品（2021年），采用目前比较流行的技术和模式，结合审计业务，研发的自主可控，保证全部版权（除开源代码版权外）的审计综合性开发、实施平台。

## 技术路线

- 云原生  微服务架构
- 前后端分离  前端 React16.4+  后端 Java 1.8+
- 平台化  表单引擎、流程引擎、查询分析引擎、数智大屏、A8组件库...

## 核心模块

- 查询分析引擎：后端服务接口（Java）、前端查询分析展现及设计器
- 表单引擎： 后端服务接口（Java）、前端表单展现及设计器
- 流程引擎：后端服务接口（Java）、前端流程展现及设计器
- 数智大屏：后端服务接口（Java）、前端大屏、工作台及设计器

## 核心产品

- A8审计数智化平台（底座）
- 审计管理系统
- 决策支持系统
- 整改跟踪系统
- 在线作业系统
- 现场作业系统
- 数据分析系统
- 其他正在开发的系统
